# 简介

### 一、项目介绍

**1.技术栈**

Vue2、Vue-router、Vuex、Vue-Cli4、Mint-UI、axios、mockjs、better-scroll等

**2.实现功能**

* 首页（轮播图、商家列表）
* 搜索页面
* 订单页面
* 用户中心
* 商家页面
  * 购物车
  * 商品列表（左右联动滚动）
  * 评论页面
  * 商家信息页面

* 登录注册页面（手机验证码登录、用户名密码登录）

### 三、项目运行

**1.server文件夹**

* npm install
* npm run start

注：启动后台应用前需先打开mongoDB数据库

**2.client文件夹**

* npm install
* npm run serve
